Sample todo app

Features:

  • Authorization with implemented cloud history
  • Three lists: todo, inProgress, done
  • Filtering by alphabet and by creation date
  • Drag and drop
  • Notifications
  • Theme switching

Technology stack:

  • Angular
  • Ng-zorro MUI
  • Firebase (authorization/firestore)
  • Angular CDK (DragAnnDrop)
  • Localization [RU, EN]
