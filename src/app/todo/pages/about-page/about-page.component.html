Sample todo app

Features:

  • Authorization with implemented cloud history
  • Three lists: todo, inProgress, done
  • Filtering by alphabet and by creation date
  • Drag and drop (on/off)
  • Notifications
  • Theme and board view switching (on/off)
  • Localization [RU, EN]

Technology stack:

  • Angular
  • Ng-zorro MUI
  • Firebase (authorization/firestore)
  • Angular CDK (DragAndDrop)
  • Ngx-translate
